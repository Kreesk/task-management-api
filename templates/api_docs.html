Документация API

Здесь описаны endpoints для работы с задачами через REST API.

GET /api/tasks

Получить список всех задач.

Response: [{"id": 1, "title": "Купить молоко", "status": "todo"}, ...]

GET /api/tasks/<int:task_id>

Получить задачу по ID.

Response: {"id": 1, "title": "Купить молоко", "status": "todo"}

POST /api/tasks

Создать новую задачу.

Request: {"title": "Новая задача"}
Response: {"id": 2, "title": "Новая задача", "status": "todo"}

PUT /api/tasks/<int:task_id>

Обновить статус задачи.

Request: {"status": "done"}
Response: {"message": "Task 1 updated", "status": "done"}

DELETE /api/tasks/<int:task_id>

Удалить задачу.

Response: {"message": "Task 1 deleted"}
